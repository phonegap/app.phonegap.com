The PhoneGap Developer App

Creating mobile apps with PhoneGap is now even easier

Develop on your local machine then push your web content straight to the PhoneGap Developer App.
No need to resign, reinstall, or relaunch the app.

Install the CLI

Install the Phonegap CLI module using npm. An in depth guide is located here.

Download the App

Download the app for the platform you are developing on.

Pair the CLI to the App

cd into your phonegap project and start the server. The server will give you an IP address to use as input for the PhoneGap App you downloaded.

Feel the Power!

Start making changes to your assets on your machine and see the new changes appear instantly!
